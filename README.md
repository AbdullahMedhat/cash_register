# Cash Register

This app create by Ruby version v3.0.3, Rails version v7.0.1

## STEPS to run the app:-
  > Make sure you have ruby v3.0.3, rails v7.0.1
  - `bundle install`
  - `rake db:create`
  - `rake db:migrate`
  - `rake db:seed`
  - `rails s` to start the server
  - `rspec` to run all the test
